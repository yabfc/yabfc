# Yet Another Factory Calculator

## Development Setup

Install [Node.js LTS](https://nodejs.org/en/learn/getting-started/how-to-install-nodejs) and [pnpm](https://pnpm.io/installation)

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Format code
pnpm format

# Run checks
pnpm check

# Build project
pnpm build
```
